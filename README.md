☕ Urban Cafe Website
📌 Project Overview

The Urban Cafe Website is a modern and responsive web application designed to showcase a cafe’s offerings, ambiance, and services. It provides users with an engaging interface to explore menu items, services, and contact details.

🚀 Features

🏠 Home Page – Attractive landing page with cafe branding

🍽️ Menu Section – Displays food and beverage items

🛎️ Services Page – Highlights cafe services in card layout

📍 Contact Section – Location and contact details

📱 Responsive Design – Works on mobile, tablet, and desktop

🛠️ Technologies Used

HTML5 – Structure of the website

CSS3 – Styling and layout

JavaScript – Interactive elements (if used)
